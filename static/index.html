Available Tools

Get Complete Resume

getResume

Retrieve the complete resume data in structured JSON format. This tool provides comprehensive access to all resume sections including personal information, experience, education, skills, and projects.

Parameters:

  • None required

Returns:

  • Personal information (name, contact details, links)
  • Professional summary
  • Work experience with achievements
  • Education background
  • Technical skills and certifications
  • Project portfolio

Get Resume Summary

getResumeSummary

Get a formatted, human-readable summary of the resume with key highlights and current role information.

Parameters:

  • None required

Returns:

  • Formatted summary with contact information
  • Current role and company
  • Education highlights
  • Top technical skills
  • Professional summary

Search Resume Content

searchResume

Search through resume content by keyword to find specific experience, skills, or achievements.

Parameters:

  • query: Search term or keyword

Returns:

  • Matching experience entries
  • Relevant achievements
  • Technical skills matches
  • Project descriptions

Authentication Info

getAuthInfo

Get current authentication status and bearer token information.

Connectivity Test

ping

Test server connectivity and verify the MCP server is responding correctly.

Authentication

This MCP server requires OAuth authentication via Descope. To access the resume data:

  1. Visit the /authorize endpoint to initiate OAuth flow
  2. Complete authentication with Descope
  3. Use the returned bearer token to access the /sse endpoint

Troubleshooting

If you encounter issues with mcp-remote, try clearing the authentication files: 

rm -rf ~/.mcp-auth

For 401 Unauthorized errors, ensure you have completed the OAuth authentication flow.
